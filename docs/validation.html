Walk-Forward Backtest Protocol

Methodology Transparency — How Every Backtest Is Run
  • All backtests use walk-forward splits — model trained on data prior to prediction date only. Zero lookahead contamination.
  • 10¢ slippage applied to every simulated entry.
  • Minimum 3 seasons required before trusting any team ML backtest result.
  • Calibration bins must be within ±2% across all probability ranges (0–10%, 10–20%, … 90–100%).
  • Closing line (no-vig) = CLV benchmark. Pinnacle closing line is the gold standard reference.
  • Paper trading minimum: 30 days before real money on any new model.
  • All results logged publicly in the Results tab — wins and losses, no cherry-picking.
